Smiley — macOS install
======================

Smiley is not notarized by Apple yet (no Developer certificate).
macOS may show: "Smiley Not Opened — Apple could not verify..."

The app is safe and open source:
https://github.com/1tsRajuWu/Smiley

INSTALL (do this once)
----------------------

1. Open the DMG
2. Drag Smiley ONCE to the Applications folder (the alias on the right)
   — do NOT drag the Applications icon itself
   — do NOT drag multiple times (each drag adds another copy)
3. Eject the DMG (right-click the disk → Eject)
4. Open Applications → RIGHT-CLICK Smiley → Open (do NOT double-click)
5. Click Open in the dialog
6. After that, double-click works forever

Multiple Smiley apps in Applications?
  Delete extras (Smiley 2, Smiley 3, …) and keep only:
  /Applications/Smiley.app

HOW TO OPEN (first time only)
-----------------------------

See steps 4–6 above.

Still blocked?
  Terminal:  xattr -cr /Applications/Smiley.app
  Then right-click → Open once.

Or: System Settings → Privacy & Security → Open Anyway

Full guide: https://github.com/1tsRajuWu/Smiley/blob/main/INSTALL-MAC.md
Only one Smiley.app per DMG — drag once to Applications. Delete Smiley 2.app or other duplicates if Finder created them.
