Features

Built to feel premium without feeling heavy.

Smiley keeps the page fast and the product flexible: static gradients, transform-only interactions, local-first presence updates, and a wider set of everyday activities than just gaming or music.

🎮

Gaming presence

Share maps, agents, squads, rank context, and game states directly on Discord with richer local data.

💻

Coding sync

Show which editor or AI tool is active, what file you are editing, and when you have gone idle.

🍜

Everyday categories

Food, chilling, anime, and studying fit naturally beside gaming and music, so your profile feels personal all day.

🎬

Animated personality

Anime GIFs, branded states, and custom activities keep the app expressive without complicating setup.

Preview

A landing page tuned for clarity, contrast, and mood.

The redesign shifts from bright orange-heavy styling to a deeper luxury palette with gold, lilac, and graphite tones while keeping the interaction model lightweight.

Ultra-premium color system

Warm gold accents, soft violet highlights, and darker graphite panels create a more expensive look while preserving readable contrast.

Performance guardrails

No blur-heavy overlays or expensive looping effects. Motion stays on transform and opacity only for a cleaner, faster feel.

Interactive category preview

The hero now behaves like a small live selector, making it obvious that Smiley supports more than just gaming, music, and coding.

🎵

Music stays polished

Album, artist, and listening presence still read clearly in the new visual system.

🍱

Food and chill states belong

The grid now covers relaxed and everyday activities so the hero better reflects the app’s full personality.

✨

Anime still has space

Anime is now one of the visible first-class categories instead of being implied only in text lower on the page.
