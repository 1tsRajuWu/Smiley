## Smiley v7.9.19 — Reinstall only for major updates

**Release date: July 8, 2026**

### Behavior

- **Major versions** (e.g. 7 → 8): show install / restart prompt.
- **Patch & minor** full builds: download silently and apply on quit — no “Restart to update” banner.
- **Live UI patches** (`src/` CSS/layout): still soft-apply without any installer flow (from v7.9.18+).
- Manual **Check for Updates** still shows status when you ask.

### Quality

- `node --check` on main / live-ui patch client
- Major-bump helper checks
