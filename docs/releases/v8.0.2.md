# Smiley v8.0.2 — GIF, settings, perf, icons

Bug-fix release after v8.0.1. Builds on the macOS install fix from 8.0.1.

## What changed

- **GIF context** — Custom Tenor GIF activities use the correct Discord activity context again
- **Idle GIF save** — Idle GIF selection persists reliably in settings
- **Performance** — Lighter presence/activity polling and UI updates
- **Branding** — Correct Smiley app, tray, and bundle icons (synced across Tauri assets)

## macOS install

Same as v8.0.1: if Gatekeeper blocks first launch, use **Open Anyway** in Privacy & Security or:

```bash
xattr -cr "/Applications/Smiley v8.app"
```

## Windows note

If SmartScreen warns on first run: click **More info** → **Run anyway**. Smiley is not yet EV code-signed (same as v7).
