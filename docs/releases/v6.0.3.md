## Smiley v6.0.3 🎉

**Patch release** — privacy consent gating and launch polish.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's fixed

- 🔒 **Install telemetry consent** — heartbeat no longer fires before the user accepts the privacy notice on first launch
- 📋 **Privacy modal** — Escape/backdrop can no longer falsely dismiss consent or leave tracking in an inconsistent state; user must tap Continue
- 📝 **Release notes** — GitHub release descriptions now populated from `docs/releases/`

### Includes v6.0.2

- Blank window fix (`body.app-ready` FOUC guard)
