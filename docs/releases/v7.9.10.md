## Smiley v7.9.10 — Valorant phase detection rebuild

**Release date: July 8, 2026**

### Fixes

- **Valorant lobby / queue / agent select / in-match** rebuilt from scratch.
- Flattens Riot nested chat presence (`matchPresenceData` / `partyPresenceData`) so `sessionLoopState` and `partyState` are actually read.
- Queue detection uses **`partyState` matchmaking**, not sticky `provisioningFlow=Matchmaking` (which lingered into agent select and live matches).
- Always probes local `/pregame` and `/core-game` APIs; local match truth wins over stale chat.
- Presence strings: **In lobby** → **Queue** → **Agent Select** → match details (never “Queuing” stuck mid-match).
