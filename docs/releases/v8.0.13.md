# Smiley v8.0.13 — Valorant presence hotfix

Follow-up to v8.0.12.

## Fixes
- **Agent select roster**: AllyTeam array shape + correct ally/enemy seats (no longer forces everyone to Ally).
- **Discord takeover**: Valorant live board always updates Discord when enabled — not only when a gaming GIF slot is active.
- **Chat score fallback**: Core-game probe passes chat presence scores when local Teams[] is empty mid-match.
- **Chat agent fallback**: Selected agent from Riot private presence when pregame/core lag.
- **valorant-api.com maps cache**: Daily refresh augments static catalog (UUID → display name, icons).
- **Large image text**: Discord hover shows friendly map name (e.g. The Range, Kasbah).

## Safe by design
- Local Riot lockfile + 127.0.0.1 only — no inject, no Tracker, no memory reads.
