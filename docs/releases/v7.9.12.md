## Smiley v7.9.12 — Live-sync toggles & nested-label click fix

**Release date: July 8, 2026**

### Fixes

- **Buttons / live-sync clicks** — removed invalid nested `<label>` markup on Aurora feature cards and gaming presence rows (clicks were flaky or inert in Chromium).
- **Music / gaming / coding toggles** — dashboard and settings toggles now persist via `saveConfig` immediately (no longer require Save & Connect); config hydrates the panel on startup.
- **Settings open crash guard** — null-safe settings button / toggle reads so a missing DOM node cannot wipe the rest of `init()`.
- **Background poll handoff** — game + coding sync remember background mode and apply it when their poll services start (window hidden/minimized).

### Security / perf

- Poll intervals unchanged and non-aggressive (music ~8s mac / 3s others; gaming/coding similar with slower background rates).
- `npm run security-check` 21/21 · no `eval` / `new Function` in app code; `spawn` limited to known media/game OS helpers.
