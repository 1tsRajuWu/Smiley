## Smiley v6.0.2 🎉

**Patch release** — fixes blank window on launch (FOUC guard).

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's fixed

- 🪟 **Blank window** — `body.app-ready` is now applied at renderer init so the FOUC guard no longer leaves a dark invisible window
- 🛡️ **Init fallback** — if `init()` throws, `app-ready` is still added so the UI remains visible
