# Smiley v8.0.1 — macOS install fix

Fixes **"Smiley v8 is damaged and can't be opened"** on macOS Sequoia / Apple Silicon.

## What changed

- Full **ad-hoc bundle codesign** (not just linker-signed binary) so Gatekeeper no longer treats the app as corrupted
- macOS CI now verifies `codesign --verify --deep --strict` before upload

## Install (macOS)

1. Download the `.dmg` for your Mac (Apple Silicon or Intel).
2. Open the DMG and drag **Smiley v8** to Applications.
3. First launch: macOS may say the developer can't be verified — open **System Settings → Privacy & Security → Open Anyway**, or right-click the app → **Open** once.

Terminal one-liner (removes quarantine after install):

```bash
xattr -cr "/Applications/Smiley v8.app"
```
