## Smiley v7.9.16 — Valorant DM / TDM / new maps

**Release date: July 8, 2026**

### Fixes

- **Map resolution** — asset paths use internal codenames (`Plummet`→Summit, `Jam`→Lotus, `HURM_*`→District/Kasbah/Drift/Glitch/Piazza). New map Summit + TDM maps resolve correctly for name and Discord map icons.
- **Team Deathmatch** — queue id `hurm` (legacy `onefa`) maps to “Team Deathmatch”; no more unknown mode / wrong map slug.
- **Deathmatch scores** — no fake round scores from chat presence; shows kills (e.g. `14 kills`) with agent + map.
- **Queues** — Retake (`fortcollins`), Spike Rush (`spikerush`), Escalation (`ggteam`), Skirmish, Knockout, and related mode icons updated from valorant-api / queue registries.
- **Unknown maps** — graceful humanized fallback instead of crashing or blanking presence.

### Quality

- `npm run game-presence-check` 143/143 (twice)
- `npm run security-check` 21/21
- `node --check` on touched modules
