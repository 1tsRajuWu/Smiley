## Smiley v7.0.0 — Final major release

**Major release** — Aurora UI (v3), Mac in-app update fix, performance & size improvements.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's new

- **Aurora UI (v3)** — Picker-first layout with preview dock on the right, list-style activities, circular character preview, underline category rail. Default for new installs; v1/v2 still in Settings.
- **Mac update fix** — In-app zip downloads no longer stuck at 0% (missing updater helpers restored; stall detection covers 0%).
- **Low-end PC performance** — Slower now-playing polls, debounced activity grid renders, lazy GIF picker thumbs, reduced backdrop blur in v3, `backgroundThrottling` unchanged.
- **Smaller desktop build** — Excludes mobile, native C# project, legacy `styles.css`, docs, and scripts from the packaged app.

### What's fixed

- Mac `download-progress` and `ensureMacUpdateDownloaded` no longer throw `ReferenceError`
- Stall timer fires when download never starts (0% hang)
- HTTP zip fallback resets stall timer during transfer
