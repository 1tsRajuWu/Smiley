## Smiley v7.9.23 — Steam / CS2 Rich Presence (not Discord streaks)

**Release date: July 8, 2026**

### About Discord “Streak” badges

**Smiley cannot add or fake Discord streak badges.** The “22x Streak” (and similar) on Recent Activity are Discord’s own Game Activity / Recent Activity feature. Smiley only drives **live Rich Presence** (logo, title, status lines). This release improves that RP for Steam and other non-Valorant games — it does **not** invent streaks.

### Fixes

- **CS2 / Steam “In the zone”** — live game sessions no longer inherit the idle Gaming catalog state. Discord now shows **game title + Playing** (or map/mode when the window title exposes it), with a **Steam capsule** as `large_image`.
- **Steam metadata stuck after first poll** — AppID + artwork are re-applied from the TTL cache on every resolve. Sessions are rebuilt from the foreground each tick; skipping lookup when the title key matched dropped logos and left Smiley fallback art.
- **Immediate AppID attach** — known processes (`cs2` → **730**, etc.) get alias AppIDs + capsule URLs **before** network lookup, so the first RPC isn’t empty.
- **Multi-game logos** — Fortnite / Overwatch / Roblox / Minecraft use shared CDN `GAME_LOGOS` (Twitch box art where Epic/Blizzard hotlinks block). Steam titles keep light `capsule_231x87` URLs (validated + cached).
- **Optional Steam small_image** — lightweight Steam community mark on Steam sessions; Valorant agent/map overlays unchanged.

### Space / APIs

- URL-only presence (no bundled game art).
- Steam store search + AppID aliases; memory + disk TTL cache under userData (unchanged max 200 entries).
- No new paid API keys.

### Limits (intentional)

- No Discord streak badges (platform feature).
- No in-match CS2 map/score without reading game memory / kernel cheats — Smiley stays local/safe (window + process + public Steam CDN only).
- Unknown non-Steam titles still fall back to Smiley logo + Playing.

### Example CS2 Discord payload

```text
details:        Counter-Strike 2
state:          Playing
large_image:    https://cdn.cloudflare.steamstatic.com/steam/apps/730/capsule_231x87.jpg
```
