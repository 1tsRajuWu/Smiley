## Smiley v7.9.13 — Four UIs + premium Aurora rebuild

**Release date: July 8, 2026**

### What’s new

- **Aurora (v3) rebuilt** — replaced the broken 3-column dashboard with a premium 2-column shell (live preview dock + activity card grid). Discord preview card restored; buttons and toggles work again.
- **Pulse (v4)** — new amber–emerald interface skin. Settings → Appearance now offers **four** complete UIs: Classic (v1), New UI (v2), Aurora (v3), Pulse (v4).
- **Live sync dock** — compact Music / Gaming / Coding toggles on the preview dock (all skins); choices persist immediately.
- **Preview modes** — Activity · Listening · Gaming · Coding chips with working hit targets across every interface version.

### Fixes

- Removed cramped left control panel / nested gaming toggle wall that fought the layout and deadened clicks.
- Shared chrome CSS for preview chips + live sync so Classic and New UI still render the dock correctly.
- Config + IPC accept and persist `uiVersion` `v1`–`v4` (no longer clamped to v3).

### Security / quality

- `npm run security-check` 21/21 · no `eval` / `new Function` in app code.
- Same presence sync backends; UI-only release for stability.
