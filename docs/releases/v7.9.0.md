## Smiley v7.9.0 — Gaming Rich Presence UI

**Release date: July 8, 2026**

Made by [@1tsRaj](https://github.com/1tsRaj).

### Gaming Rich Presence

- **Settings panel** — new Gaming Rich Presence section with per-field toggles (mode, party, agent, score, rank, map art, elapsed time, K/D)
- **Discord preview** — Smiley-branded live preview card with state pills (Lobby, Queue, Agent Select, In Match, Deathmatch)
- **Presence formatting** — pipe-separated state lines per game phase; respects toggle options
- **Discord images** — large map/mode art + small agent or rank badge overlay
- **Elapsed timer** — match-only green timer via Discord RPC `startTimestamp`
- **Optional Riot API key** — encrypted local storage for Valorant rank on Discord

### Website

- Fresh orange + navy landing page redesign
- Music sync and Gaming Rich Presence hero features
- Discord-style presence mockups

### Looking ahead

**v8.0.0 will be the final planned release** of Smiley. After that, only occasional bug fixes — and a major update remains possible but unlikely. The community is welcome to fork, modify, and carry the project forward. Thank you for being here.
