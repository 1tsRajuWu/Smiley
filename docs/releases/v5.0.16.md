## Smiley v5.0.16 🎉

**Apple Music fix** — now playing works again in packaged Mac builds.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's fixed

- 🍎 **Apple Music detection** — JXA script now runs via stdin so packaged `.asar` builds can read now playing data
- 🔄 **AppleScript fallback** — if MediaRemote/JXA returns nothing, Smiley asks Music.app (and Spotify) directly
- 📦 **Packaged build fix** — JXA script is unpacked from asar for reliable macOS detection
