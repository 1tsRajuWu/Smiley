# Smiley v8.0.4

**Critical macOS fix** — v8.0.3 could crash immediately on launch when opened from Finder/DMG because the Discord client ID was only read from dev paths, not embedded in the release binary.

## Fixes

- Embed `discord.app.json` client ID at compile time so release builds start reliably on macOS (and all platforms) regardless of launch directory.
- Verified: app stays running when launched with cwd `/` (same as double-click from Applications).
