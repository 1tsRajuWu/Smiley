## Smiley v7.7.7 — Final release

**Release date: July 7, 2026 (7/7/2026)**

**The one and only final version** — settings and custom GIFs now survive restart and in-app updates.

Made by [@1tsRaj](https://github.com/1tsRaj).

### Settings & custom GIF persistence fix

- **Config backup & recovery** — every save writes `config.secure.bak`; decrypt failures try backup and legacy `config.json` before falling back
- **No silent wipe** — failed decrypt no longer overwrites your encrypted config with defaults
- **Custom GIF safety** — orphan cleanup is skipped when config fails to load, so uploaded GIFs are not deleted on a bad read
- **Keychain migration** — legacy macOS Keychain configs attempt recovery before reset
- **Update-safe flush** — settings are flushed to disk before in-app update install (Mac zip installer + Windows/Linux NSIS)
- **userData unchanged** — custom animations, activities, and wallpapers stay in the OS app-data folder across updates

### Aurora UI (v3)

- Picker-first layout with preview dock, circular character preview, underline category rail
- Cyan-violet Aurora Trails theme; reduced blur for low-end PCs

### Mac update fix

- In-app zip downloads with custom installer (no ShipIt wipe of user data)

### Instant music sync

- macOS mediaremote stream; Windows/Linux native now-playing events
- Instant Discord song updates; album art follows async

### Performance & low-end polish

- Less CPU when tray-hidden; smarter now-playing polls; fewer RPC floods
- Lighter renderer — GIFs pause when hidden; debounced grid renders
