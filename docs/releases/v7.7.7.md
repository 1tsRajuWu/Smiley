## Smiley v7.7.7 🎉

**Final performance release** — the "7 XD" send-off. Smooth as butter on low-end PCs.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's improved

- ⚡ **Less CPU when tray-hidden** — music sync pauses UI/renderer updates and slows polling while the window is minimized or hidden; Discord still updates on song changes
- 🎵 **Smarter now-playing** — macOS osascript/JXA polls less often; mediaremote stream events are debounced; progress-only updates throttled
- 📡 **Fewer Discord RPC floods** — duplicate presence payloads deduped; progress-only RPC spaced out; tray/status broadcasts debounced longer
- 🖼️ **Lighter renderer** — GIFs pause when hidden; music progress bar uses rAF batching at 2s intervals; animations freeze in background
- 🧈 **Low-end friendly** — Aurora v3 already skips heavy backdrop blur; render-paused mode stops timers and animated GIF decode
