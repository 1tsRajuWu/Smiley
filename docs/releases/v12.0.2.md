# Smiley v12.0.2

**Patch release** — fixes activity GIF tile sizing and adds visible v12 identity polish.

## What's fixed

- **GIF tile sizing** — Studio horizontal grid now sizes `.tile-wrap` (not the inner button), with square `aspect-ratio` tiles and absolutely positioned GIFs so cards no longer stretch, shrink, or crop incorrectly
- **Hero previews** — Arcade cabinet screen and all skins use consistent `hero-gif` / `tile-gif` classes with `object-fit: cover`
- **Terminal nested buttons** — unchanged click bus; tile layout no longer depends on broken flex inheritance

## What's new (v12 identity)

- Studio skin **v12 badge** in the header next to “Smiley”
- Studio-only **mint accent** (`#2ee6a8`) to distinguish v12 from the reused v8 orange palette
- Settings → App tab states **Smiley v12 — final native desktop app**
- Window title remains **Smiley**; version shown in Settings
