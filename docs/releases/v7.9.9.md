## Smiley v7.9.9 — Coding detects Cursor (lsappinfo)

**Release date: July 8, 2026**

### Fixes

- **Cursor / VS Code detection on Mac** — uses `lsappinfo` (no Accessibility permission) instead of relying only on System Events JXA.
- **Blank Discord on Coding** — presence is applied immediately when you pick Coding, then live-updates when an editor is found.
- Restored **Show current editor (Coding)** setting.
- v7.9.8 tag accidentally missed these files; this release contains the real fix.
