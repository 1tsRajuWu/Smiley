# Smiley v8.0.11

**Idle GIF settings save fix**

## Fixes

- **Idle GIF persistence:** Settings → Auto → Idle GIF now saves reliably. The UI reads the input via global `#cfgIdleGif`, always sends an explicit `idleGif` string to Rust, and Enter in the settings form triggers save. `save_config` validates Tenor URLs before sanitize and applies the user’s URL from the raw payload.
