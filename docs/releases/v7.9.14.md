## Smiley v7.9.14 — Buttons, Settings, connection status

**Release date: July 8, 2026**

### Fixes

- **Dead buttons / Settings tabs** — Controls inside the app and Settings (Appearance, Advanced, About, Cancel, Check for Updates, Save & Connect) no longer behave like inert text. Hit targets force `pointer-events: auto` and `-webkit-app-region: no-drag` so Electron drag regions cannot swallow clicks.
- **Stuck “Connecting…”** — Connection pill hydrates from main-process status on startup and status events, so it updates to Connected / Disconnected instead of staying Connecting forever.
- **Cancel in Settings** — Explicit Cancel button closes the dialog cleanly.

### Gaming (all games)

- Removed the **Riot API key** field from Settings. Valorant extras still come from the **local Riot client** when present; no developer API key is shown or required.
- Gaming presence options and preview are **general for all games** (not Valorant lobby/agent/deathmatch focused).

### Quality

- `npm run security-check` 21/21
