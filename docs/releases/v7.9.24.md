## Smiley v7.9.24 — Valid Steam mark + game logos that Discord accepts

**Release date: July 8, 2026**

Hotfix on top of v7.9.23: Steam `small_image` no longer points at a **404** community avatar (Discord showed a loading spinner). Fortnite / Overwatch / Roblox / Minecraft keep Twitch CDN box art that returns HTTP 200.

### Fixes

- **Steam small_image** — use stable Steam client capsule (`apps/753/capsule_231x87.jpg`) instead of a dead community image hash.
- **Steam large_image for CS2** — `capsule_231x87.jpg` on `cdn.cloudflare.steamstatic.com` (verified 200 / `image/jpeg`).
- **Presence text** — CS2 lobby → `In lobby`; match → `In match · Competitive`; default → `Playing` (never catalog “In the zone”).
- **Non-Steam logos** — Epic / Blizzard / rbxcdn / minecraft.net URLs that 403/hotlink-block are replaced with Twitch `ttv-boxart` JPEGs Discord’s proxy can fetch.
- **Fixtures** — `game-presence-check` / `game-assets-check` assert lobby/match CS2 text, Twitch logos, and Steam mark validation.

### Example CS2 Discord payload

```text
details:        Counter-Strike 2
state:          Playing          # or "In lobby" / "In match · Competitive"
large_image:    https://cdn.cloudflare.steamstatic.com/steam/apps/730/capsule_231x87.jpg
small_image:    https://cdn.cloudflare.steamstatic.com/steam/apps/753/capsule_231x87.jpg
```
