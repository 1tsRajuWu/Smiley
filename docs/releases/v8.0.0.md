# Smiley v8.0.0 — Native desktop (Tauri + Rust)

**Final major platform release.** Native shell, lower memory, privacy-first Valorant live board.

## Highlights

- **4 skins** — Studio, Arcade, Terminal, Zen
- **Custom GIF activities** — Tenor HTTPS + your own presets
- **Live Valorant board** — ally/enemy agents, score, KDA (local Riot API only — no Tracker.gg, no injectors)
- **Music presence** — Spotify / Apple Music while “Listening” is selected
- **Privacy defaults** — other players’ Riot IDs off until you opt in (Settings → Privacy)
- **Animated wallpaper** — pauses in tray for CPU
- **Donate** — optional PayPal tip (`paypal.me/1tsRaj`), separate from Discord

## Install

1. Download the artifact for your OS below.
2. Open **Discord desktop** (required for Rich Presence).
3. On first run, Smiley uses the bundled Discord Application Client ID.

## macOS

If Gatekeeper blocks the app: right-click → Open once, or allow in System Settings → Privacy & Security.
