## Smiley v7.0.3 🎉

**Patch release** — near-instant now-playing sync on macOS and no Discord green timer for music.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's fixed

- ⚡ **Track changes in ~1–2s** — macOS polls every ~1.2s while listening; fast 350ms re-poll right after title/artist changes
- 🛑 **No more ~8s gap** — removed aggressive 5s-per-failure JXA backoff that stacked on top of the 3s poll interval
- 🎵 **Text-first Discord updates** — song title/artist push immediately; album art follows async
- 🚫 **No green elapsed timer** — listening presence never sends `startTimestamp` (even during art-only updates)
