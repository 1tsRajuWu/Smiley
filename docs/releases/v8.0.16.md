# Smiley v8.0.16 — GIF card click hotfix

Patch release for broken activity/GIF card selection in the v8 UI.

## Fixes
- **Reliable card apply**: activity tiles now use a dedicated apply button instead of nested interactive controls, so clicking a GIF/activity card consistently sets the selected presence again.
- **Favorites no longer steal apply clicks**: the star and custom delete affordances are now separate real buttons, which removes the invalid nested-button behavior that made clicks flaky after recent layout changes.
- **Overlay no longer blocks cards**: the live Valorant board is now pointer-transparent, so it cannot intercept card clicks when rendered above the board area.

## Verification
- `npm run build`
- `cargo check`

## Release relationship
- `v8.0.15` remains the updater reliability patch already in progress elsewhere.
- `v8.0.16` stacks this UI hotfix on top so release notes and versioning do not collide.
