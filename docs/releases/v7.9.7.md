## Smiley v7.9.7 — Coding sync detects Cursor again

**Release date: July 8, 2026**

### Fixes

- **Coding sync on macOS** — frontmost-app detection used a broken JXA `byAttribute('frontmost')` query that usually returned empty, so Cursor / VS Code / Claude never appeared on Discord.
- Uses `applicationProcesses.whose({frontmost: true})` and, when Smiley itself is focused, falls back to a running coding editor (Cursor, VS Code, etc.).
- Broader app matching (bundle IDs + process names) and faster coding poll interval.
