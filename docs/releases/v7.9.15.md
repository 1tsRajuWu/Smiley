## Smiley v7.9.15 — Valorant logo + agent on Discord

**Release date: July 8, 2026**

### Fixes

- **Discord large image** — always the official Valorant red **V** logomark (Riot CDN). No longer uses the Standard gamemode / map geometric icons as `large_image`.
- **Agent on presence** — Agent Select and in-match show the selected agent when local pregame/core-game returns a `CharacterID`.
- **Pregame player parse** — correctly reads `AllyTeam.Players` (was treating `AllyTeam` as a flat array, so CharacterID never resolved).
- **Presence copy** — Agent Select: `Jett · Agent Select` + mode/map/party on state; In match: `Jett · Ascent` details with score/party/mode on state.
- **small_image** — agent icon preferred; map/mode fallback.

### Quality

- `npm run game-presence-check` 93/93 (twice)
- `npm run security-check` 21/21 (twice)
- `node --check` on touched modules
