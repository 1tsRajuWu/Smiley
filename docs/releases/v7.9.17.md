## Smiley v7.9.17 — UI layouts + Settings reliability

**Release date: July 8, 2026**

### Improvements

- **Distinct UI layouts** — Classic / Studio / Aurora / Zen layout structure is clearer so switching Appearance themes feels meaningfully different.
- **Settings open reliability** — opening Settings no longer fails silently if version labels are missing; shows an error toast instead.
- **Dev live reload** — `npm run dev` auto-refreshes the UI on `src/` edits and restarts when main/preload/electron change (packaged installs unchanged).

### Quality

- Packaged auto-update path unchanged — installed apps pick this release up on next launch / Check for Updates.
