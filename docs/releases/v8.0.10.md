# Smiley v8.0.10

**Instant music sync (MediaRemote) + v8.0.8 GIF/coding fixes**

## New

- **Instant music sync (macOS):** While **Listening to music** is active, Smiley uses the **mediaremote-adapter** stream (same technique as [Music Presence](https://musicpresence.app) and v7) for near-instant track changes. osascript fallback if the adapter is unavailable.
- **Linux MPRIS:** Spotify, VLC, and other MPRIS2 players via D-Bus while listening.

## Includes v8.0.8

- **Music banner GIF on Discord** — animated Tenor GIF as `large_image` for listening presence.
- **Live coding detection** — foreground editor + window title on macOS (Cursor, VS Code, Xcode, JetBrains, terminals).

## Technical

- Bundles `electron/mediaremote-adapter/` in macOS app resources (from [ungive/mediaremote-adapter](https://github.com/ungive/mediaremote-adapter)).
- MediaRemote stream idles until metadata changes; thread sleeps 8s when not on the listening activity.
