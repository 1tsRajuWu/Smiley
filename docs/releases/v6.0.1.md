## Smiley v6.0.1 🎉

**Patch release** — fixes settings export IPC and Discord RPC login cleanup.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's fixed

- 📤 **Export settings** — `exportSettings` exposed in preload so Advanced → Export settings (E2EE) works again
- 🔌 **Discord RPC login errors** — failed logins now destroy the client instead of leaving orphaned IPC sessions
