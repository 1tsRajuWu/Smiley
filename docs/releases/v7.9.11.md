## Smiley v7.9.11 — Restore live sync UI & coding presence

**Release date: July 8, 2026**

### Fixes

- **Buttons / dashboard** — restored the stripped Aurora control panel, music/gaming/coding preview bars, and live-sync toggles that left UI half-wired after recent redesigns.
- **Category tabs set presence** — Gaming / Chill (music) / Work (coding) tabs (and Cmd/Ctrl+1–6) now select the primary live activity and push Discord presence.
- **Coding blank / double-apply** — coding sync applies the static template once on select; `ensureRunning` no longer re-pushes an empty fallback. Main no longer races a second coding apply.
- **Music blank wait** — listening presence shows the static template immediately while now-playing detection starts.
- **Background polling** — game + coding sync respect window-hidden background mode (slower polls when minimized/hidden).
- **Riot settings** — panel “Riot API key & more” + Save key listeners restored.

### Security

- Malware-pattern review: no `eval` / `new Function` / obfuscated payloads in app code.
- Outbound hosts are expected product endpoints (Discord/GitHub/Riot/Steam/iTunes/media APIs).
- `npm run security-check` 21/21 · `npm audit --omit=dev --audit-level=critical` clean.
