# Smiley v8.0.18 — Simplified Valorant presence

Patch release removing the in-app ally/enemy roster board and simplifying Valorant Discord presence to self-only details.

## Changes
- **Removed live ally/enemy roster board** — no more floating LIVE MATCH panel with team columns in-app.
- **Simplified Valorant Discord presence** — map, mode, your agent, score, and logos only; no other-player roster lines.
- **Settings cleanup** — removed match-board and other-player privacy toggles; Full vs Minimal Valorant detail stays under Discord.

## Verification
- `cargo check`
- `cargo test`
- `npm run build`

## Privacy / scope
- Local Riot lockfile + `127.0.0.1` APIs only
- Public Valorant catalog/art metadata only
- No inject, no memory reads, no remote roster APIs
