## Smiley v7.0.1

**Patch release** — removes the green elapsed timer on Discord for listening / now-playing music.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's fixed

- **No green timer for music** — Discord no longer shows an elapsed-time badge while listening; song title and artist only
- **Preview timer hidden** — in-app preview timer is hidden for the listening activity
- **Session state cleared** — listening sessions no longer carry over a `startTimestamp` from other activities

**Tip:** If you still see the green timer after updating, tap **Listening** again (or clear and re-set your status) so Discord gets a fresh presence without timestamps.
