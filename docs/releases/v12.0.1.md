# Smiley v12.0.1

**Patch release** — fixes macOS CI signing (repo-root entitlements) so Apple Silicon and Intel DMGs build and ship.

## What's fixed

- **macOS entitlements** — `build/entitlements.mac.plist` at repo root for Tauri bundle signing (was missing at v12.0.0 tag, Mac matrix jobs failed)
- All platforms rebuilt with version **12.0.1**
