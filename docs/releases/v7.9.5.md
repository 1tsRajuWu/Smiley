## Smiley v7.9.5 — Complete multi-platform release

**Release date: July 8, 2026**

### Fixes

- **Windows auto-update / downloads** — GitHub Releases are kept as drafts until macOS, Windows, and Linux artifacts (including `latest.yml` + `Smiley-Setup-*.exe`) are all uploaded, then published. This prevents `/releases/latest` from going live with mac-only assets (the v7.9.4 issue).
- Includes the Aurora UI simplify work from v7.9.3 (preview dock / Valorant copy cleanup) plus telemetry fixes from v7.9.4.
