# Smiley v8.0.5

**Close-to-tray + idle GIF persistence** — matches Electron v7 behavior for window close, and fixes custom idle GIF URLs reverting to default after save/reload.

## Fixes

- **Close → tray (macOS, Windows, Linux):** Window close (X / red button) hides to tray instead of quitting when “Minimize to tray” is on (default). Quit is tray menu only. Builder-level `CloseRequested` + `ExitRequested` handlers; macOS dock hides while in tray.
- **Idle GIF save:** Sanitize no longer silently resets valid custom Tenor URLs to default; protocol-less CDN paths get `https://` prepended; invalid URLs return a clear settings error instead of reverting; disk roundtrip tests added.
