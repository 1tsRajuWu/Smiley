## Smiley v6.0.0 🎉

**Major stability release** — fixes critical runtime bugs, hardens security, and improves performance across the desktop app.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's fixed

- 🎵 **Now playing progress** — playback position and Discord music timestamps resync correctly during long tracks
- 🖼️ **Wallpaper cache** — switching wallpapers no longer shows a stale image from the previous file
- 🔌 **Discord RPC connect** — timed-out connections no longer leave orphaned IPC sessions
- 📋 **Activity grid** — selection no longer corrupts the grid render cache key
- 🛡️ **Path traversal** — user media paths validated with `isPathInsideResolvedDir` across wallpapers, GIFs, and custom animations
- 📥 **Settings import** — rejects non-object JSON to prevent corrupt imports
- 🍎 **macOS close-to-tray** — window visibility state syncs on minimize, hide, restore, and show
- ⚡ **Performance** — debounced window-state saves, session-stats writes, and status broadcasts; background throttling enabled
- 🔄 **Auto-updater** — resets quit flag if `quitAndInstall` fails so the app stays usable

### What's improved

- Centralized `sendToWindow` helper for safer IPC to renderer
- `flushPendingDiskWrites` on close-to-tray and quit for reliable config persistence
- Duplicate `onSelectActivity` listener removed
