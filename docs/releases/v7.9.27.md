## Smiley v7.9.27 — Valorant agent presence fix

**Release date: July 8, 2026**

### Fixes

- **Agent on Discord** — in-match and Agent Select now resolve your agent from more live Riot API shapes (`CharacterID`, `AgentID`, `PlayerIdentity.CharacterID`, `Character` UUID string, puuid-keyed `Players` maps).
- **Pregame → match handoff** — when core-game returns a match without an agent yet, Smiley keeps the agent from pregame instead of dropping it.
- **Offline agent names** — bundled UUID→name map so `Jett · Ascent` still works when valorant-api.com is unreachable; `small_image` agent icon unchanged.
- **UI sanitization** — `agentId` is now preserved in the renderer game-session snapshot (still no PUUIDs/tokens).

### Also in this patch

- Foreground game switch forces an immediate presence refresh (CS2 vs Valorant handoff).

### Quality

- `npm run game-presence-check` (×2) — 234 checks
- `npm run security-check` — 21 checks
