## Smiley v5.0.0 🎉

**Major release** — Discord Rich Presence with animated anime GIFs. All platforms synced, UI polished, and advanced power-user features included.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's new in v5

#### Advanced presence
- **Activity profiles** — Save favorite combos as named presets; apply from Settings or the tray menu (up to 8 profiles).
- **Auto-rotate favorites** — Cycle starred activities every 5–120 minutes.
- **Pause / resume** — `Cmd/Ctrl+Shift+P` or tray menu hides Discord status without losing your pick.
- **Favorite hotkeys** — `Cmd/Ctrl+Shift+1–9` instantly applies favorite #1–9.
- **Session stats** — Tracks total time per activity in Settings → Advanced.

#### UI & polish
- Cleaner v2 layout: improved spacing, contrast, and settings organization.
- Pause state shown in the connection pill.
- Advanced settings grouped into clear sections (Profiles, Automation, Data).
- Update banner and footer visual refinements.

#### Reliability & platforms
- **Mac in-app updates** — Fixed GitHub redirect downloads; zip fallback for ad-hoc builds.
- **GIF loading** — Broader CSP for Tenor/Giphy subdomains; custom upload paths restored.
- **GitHub Releases** — Versioned download URLs, full release notes on every tag, mobile APK after desktop CI.
- **All platforms synced** — Windows, macOS (Intel + Apple Silicon), Linux, Android companion at **v5.0.0**.

### Before you install

- Discord **desktop** app open (not Invisible)
- **OS:** macOS 11+, Windows 10 (1809+) / 11, Linux Ubuntu 20.04+ — [requirements](https://github.com/1tsRajuWu/Smiley/blob/main/docs/MINIMUM-REQUIREMENTS.md)

- **Mac v4.1.9+** — Use **Install update** in the app banner, or download the DMG once if updates fail.
- **Windows / Linux** — Download the new installer; settings and custom GIFs are preserved.
