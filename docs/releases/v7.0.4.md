## Smiley v7.0.4 🎉

**Patch release** — instant music sync like Music Presence, plus an in-app progress bar.

Made by [@1tsRaj](https://github.com/1tsRaj).

### What's new

- ⚡ **Instant track changes** — macOS uses `mediaremote-adapter` event stream (same technique as [Music Presence](https://musicpresence.app)); no more 1.2s poll lag
- 🎵 **Music preview bar** — album art, title, artist, and a smooth in-app progress bar while Listening to music is active
- 🔄 **Smart fallbacks** — native `node-nowplaying` events on Windows/Linux; JXA poll only when the adapter is unavailable on Mac
- 🚫 **No Discord green timer** — listening presence still never sends `startTimestamp`
