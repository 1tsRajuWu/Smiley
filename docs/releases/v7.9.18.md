## Smiley v7.9.18 — Silent signed UI live patches

**Release date: July 8, 2026**

### What's new

- **Silent live UI** — after this install, CSS / layout / renderer changes you push to GitHub can land on users without an installer “Restart to update” flow.
- **Signed overlays** — apps download a zip from GitHub Pages, verify **ed25519 + SHA-256**, extract to a local `live-ui` folder, then soft-reload the window.
- **Privacy** — patch checks are anonymous HTTPS GETs only; offline users keep the last good UI. Discord RPC / encryption / main-process updates still use the normal release channel.

### For you (author)

- Edit `src/` → push `main` → Pages CI publishes `/live/`
- Secret required: `SMILEY_LIVE_PATCH_PRIVATE_KEY` (already documented in `docs/RELEASING.md`)

### Quality

- Local signature roundtrip verified
- `npm run security-check` 21/21
