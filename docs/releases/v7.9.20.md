## Smiley v7.9.20 — Steam / multi-game Discord logos (light CDN)

**Release date: July 8, 2026**

### Fixes

- **Broken Discord `large_image` spinner** — unknown / Steam games were falling back to a Tenor *loading* GIF (`GAMING_FALLBACK`). That is what users saw as a spinning image for Counter-Strike 2 and other non-Valorant titles.
- **Steam games** — resolve AppID (aliases + Steam store search), then use a **light Steam CDN capsule** (`capsule_231x87.jpg`, ~10KB) validated before RPC push. Prefer capsules over heavy `library_600x900` / hero art.
- **Unknown games** — Smiley logo URL (or omit spinner) — never send a 404 or loading GIF.
- **Valorant unchanged** — still the official **128×128** Riot CDN V logomark; agent/map overlays stay on `media.valorant-api.com` as remote URLs only.

### Space / weight

- No game art bundled. Presence carries **URL strings only**.
- AppID → artwork cache in memory + light disk file under userData (TTL + max 200 entries).
- 128px Valorant logo + Steam capsules keep Discord’s proxy downloads small without packing millions of assets into the app.

### Quality

- `npm run game-presence-check` 158/158 (×2)
- `npm run game-assets-check` 17/17 (×2)
- `npm run security-check` 21/21
- `node --check` on touched modules
