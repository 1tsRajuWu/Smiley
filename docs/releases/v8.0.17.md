# Smiley v8.0.17 — Riot local freshness upgrade

Patch release for faster, more reliable local-only Valorant match details in Smiley v8.

## Fixes
- **Faster live updates while Valorant is active**: the local Riot probe now uses an adaptive cadence, refreshing much more often during live matches and agent select while staying slower in lobby or when music/coding own the Discord slot.
- **Better local scoreboard coverage**: Smiley now merges the local `/core-game` match payload with the local scoreboard payload, improving score and KDA freshness without leaving the machine.
- **Safer cache behavior**: Riot probe caching is now phase-aware, so active matches stop waiting on the old coarse cache window before the board and Discord presence can update.

## Verification
- `cargo check`
- `cargo test`
- `npm run build`

## Privacy / scope
- Local Riot lockfile + `127.0.0.1` APIs only
- Public Valorant catalog/art metadata only
- No inject, no memory reads, no tracker-style remote calls
