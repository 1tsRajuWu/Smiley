## Smiley v7.9.22 — Sticky gaming + Steam CDN logos

**Release date: July 8, 2026**

Ships the gaming fixes that landed on `main` as v7.9.20 / v7.9.21 but never published a complete Latest release (orphaned drafts / missing macOS assets).

### Fixes

- **Steam / AppID logos via light CDN** — resolve Steam AppID, then use a validated light capsule (`capsule_231x87.jpg`) for Discord `large_image` (not heavy library art or spinner URLs).
- **128×128 Valorant CDN** — logo / agents prefer compact Valorant CDN art Discord can proxy reliably.
- **Out-of-focus games stay on Discord** — focusing Discord, Chrome, or other ignored apps no longer clears gaming presence while the game process is still running (e.g. Counter-Strike 2 / Valorant).
- **Still prefers focused games** — bringing a different known game to the foreground switches presence immediately. With two games running and neither focused, Smiley keeps the last sticky title rather than thrashing.

### How sticky detection works

1. Poll foreground window (unchanged rate: ~5–8s, slower when Smiley is backgrounded).
2. If the front app is a game → use it.
3. If the front app is Discord/browser/Steam client/etc. → scan running processes for known games; keep the previous game if still alive, or adopt the only known game running.
4. Presence clears only when that process exits (or you focus a different known game).
