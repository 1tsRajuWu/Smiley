# Smiley v12.0.0

**Smiley 12** — the final desktop app. Tauri + Rust, four polished skins, stable core.

## What's new

- **V12 product line** — no more v8.0.x patch confusion; version **12.0.0** everywhere
- **Clean architecture** — single UI click bus, settings that save, tray close behavior
- **Best of v7 + v8** — MediaRemote music, Valorant local API, CS2/game probe, in-app updater
- **4 skins** — Studio, Arcade, Terminal, Zen
- **Valorant Discord** — map, mode, self agent, score, logos — **no ally/enemy roster board**
